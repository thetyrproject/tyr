# Appendix C — Glossary

## Backend

A compiler component that translates Tyr into a target representation.

---

## Clock

A synchronization source used by synchronous hardware.

---

## Compiler Directive

A source-level instruction processed before language analysis.

---

## Concurrency

The simultaneous existence and operation of hardware components.

---

## Elaboration

The process of constructing the complete hardware hierarchy.

---

## Event

An asynchronous trigger used by event-driven hardware.

---

## Flow Block

A behavioural block representing continuously active hardware.

---

## Hardware Graph

An implementation-defined internal representation of a hardware design.

---

## Module

The fundamental reusable hardware design unit.

---

## Signal

A hardware communication path.

---

## Translation Unit

A single Tyr source file after compiler directives have been processed.

---

## Type

A language construct defining the representation and permitted operations of hardware objects.
