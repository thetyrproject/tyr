# Appendix B — Reserved Keywords

The following identifiers are reserved by the Tyr language.

```
after
array
assert
assume
bit
bus
clock
const
cover
default
delay
else
end
event
flow
if
inout
input
match
memory
module
output
register
return
signal
tand
tnot
tor
trit
txor
use
wait
```

Future reserved words

```
enum
generic
interface
namespace
package
struct
union
```

Programs shall not redefine reserved keywords.
