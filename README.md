# Tyr

> **One Language. Every Digital Architecture.**

Tyr is a modern hardware description language designed for synchronous, asynchronous, binary, and multi-valued digital systems.

Unlike traditional HDLs, Tyr is designed around hardware semantics rather than implementation technology.

The Tyr repository contains the official language specification.

The reference compiler is developed separately in the `tyrc` repository.

---

## Repository Structure

- `docs/specification/` — Official language specification
- `docs/examples/` — Reference programs
- `docs/decisions/` — Language design decisions
